## What is Wireframing?
Wireframing is the process of creating a **visual blueprint** or **skeletal framework** of a website, application, or digital product. It outlines the basic structure, layout, and functionality of a design before any actual development or detailed visual design begins. 

In simple terms, a wireframe acts as a **guide** that shows where key elements like navigation bars, buttons, images, and text will be placed on each page or screen.
